\# FinSight Lanka Customer Analytics



\## Project Overview

This project was completed as part of an Analytics \& AI Trainee assessment. The objective was to analyse customer financial behaviour, loan risk, digital engagement, and cross-sell opportunities using a fintech customer dataset.



The analysis focused on transforming raw customer data into meaningful business insights through data cleaning, exploratory analysis, KPI tracking, and dashboard visualization.

\## Key Tasks Performed



\### Data Cleaning

\- Handled missing values

\- Removed duplicate records

\- Standardized inconsistent text values

\- Converted incorrect data types

\- Created derived analytical columns



\### Customer Savings Analysis

\- Analysed savings behaviour by:

&#x20; - Customer Segment

&#x20; - Province

&#x20; - Age Group

&#x20; - Mobile App Usage



\### Loan Portfolio \& Risk Analysis

\- Calculated Debt-to-Savings Ratio

\- Identified high-risk customers

\- Analysed loan repayment behaviour

\- Evaluated loan penetration rates



\### Digital Engagement Analysis

\- Compared mobile app users vs non-users

\- Identified cross-sell opportunities

\- Evaluated acquisition channel performance



\### KPI Dashboard

Developed a Power BI dashboard containing:

\- Total Customers

\- Average Savings Balance

\- Customer Segment Insights

\- Mobile App Usage Analysis

\- Loan Repayment Analysis



\---



\## Key Insights

\- Younger customers demonstrated strong savings behaviour.

\- Mobile app users maintained significantly higher savings balances.

\- Certain customer segments showed elevated repayment risk.

\- Multiple opportunities existed for cross-selling financial products.
